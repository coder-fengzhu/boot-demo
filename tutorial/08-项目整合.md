## 8.1 完整调用流程

### 查询单个学生的完整流程

```
┌────────────────────────────────────────────────────────────────────────┐
│                        GET /student/1 查询流程                          │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  1. 浏览器发送：GET http://localhost:8080/student/1                     │
│                              ↓                                         │
│  2. Tomcat 接收请求，路由到 StudentController                           │
│                              ↓                                         │
│  3. @GetMapping("/student/{id}") 匹配                                  │
│     @PathVariable 提取 id = 1                                          │
│                              ↓                                         │
│  4. Controller 调用: studentService.getStudentById(1)                  │
│                              ↓                                         │
│  5. Service 执行:                                                      │
│     - studentRepository.findById(1) → 返回 Optional<Student>           │
│     - student.orElseThrow() → 得到 Student 实体                        │
│     - StudentConverter.convertStudent() → 转换为 StudentDTO            │
│                              ↓                                         │
│  6. Controller 调用: Response.newSuccess(dto)                          │
│                              ↓                                         │
│  7. 返回 JSON: {"success":true, "data":{...}, "errorMsg":null}          │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘
```

### 新增学生的完整流程

```
┌────────────────────────────────────────────────────────────────────────┐
│                      POST /student 新增流程                             │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  1. 浏览器发送：POST /student                                          │
│     Body: {"name":"张三","email":"zhangsan@test.com","gender":"BOY"}   │
│                              ↓                                         │
│  2. @RequestBody 将 JSON 映射为 StudentDTO 对象                        │
│                              ↓                                         │
│  3. Service 执行:                                                      │
│     - studentRepository.findByEmail() → 检查邮箱是否重复                │
│     - StudentConverter.convertStudent(dto) → DTO 转 Entity            │
│     - studentRepository.save(student) → 存入数据库                     │
│     - 返回新生成的 student.id                                          │
│                              ↓                                         │
│  4. 返回 JSON: {"success":true, "data":1, "errorMsg":null}             │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘
```

## 8.2 项目结构总览

```
boot-demo/
├── pom.xml                           # Maven 依赖配置
│
├── src/main/java/com/tutorial/bootdemo/
│   │
│   ├── BootDemoApplication.java      # 【启动类】Spring Boot 入口
│   │                                  # @SpringBootApplication 标记
│   │
│   ├── Response.java                  # 【统一响应】API 响应封装
│   │                                  # Response<T> 泛型类
│   │
│   ├── controller/
│   │   └── StudentController.java    # 【控制器层】处理 HTTP 请求
│   │                                  # @RestController + @GetMapping/PostMapping
│   │
│   ├── service/
│   │   ├── StudentService.java       # 【服务接口】定义业务方法
│   │   └── StudentServiceImpl.java   # 【服务实现】处理业务逻辑
│   │                                  # @Service + @Transactional
│   │
│   ├── dao/
│   │   ├── Student.java              # 【实体类】数据库表映射
│   │   │                              # @Entity + @Table + @Column
│   │   └── StudentRepository.java    # 【数据访问层】操作数据库
│   │                                  # extends JpaRepository
│   │
│   ├── dto/
│   │   └── StudentDTO.java           # 【数据传输对象】API 传输用
│   │
│   ├── enums/
│   │   └── Gender.java               # 【枚举】固定常量
│   │
│   └── converter/
│       ├── GenderConverter.java     # 【JPA 转换器】enum ↔ String
│       └── StudentConverter.java    # 【DTO 转换器】Entity ↔ DTO
│
└── src/main/resources/
    └── application.properties        # 【配置文件】数据库连接等
```

## 8.3 各层职责总结

```
┌─────────────────────────────────────────────────────────────────┐
│                         分层架构                                 │
├─────────────┬───────────────────────────────────────────────────┤
│  Controller  │  接收 HTTP 请求，调用 Service，返回 Response       │
├─────────────┼───────────────────────────────────────────────────┤
│   Service   │  处理业务逻辑，事务管理，调用 Repository            │
├─────────────┼───────────────────────────────────────────────────┤
│ Repository  │  数据访问，继承 JpaRepository 自动获得 CRUD 方法    │
├─────────────┼───────────────────────────────────────────────────┤
│   Entity    │  数据库表映射，JPA 注解标记                         │
├─────────────┼───────────────────────────────────────────────────┤
│    DTO      │  数据传输对象，分离内外层                           │
├─────────────┼───────────────────────────────────────────────────┤
│  Converter  │  Entity ↔ DTO 转换                               │
└─────────────┴───────────────────────────────────────────────────┘
```

## 8.4 RESTful API 总结

| 方法 | 路径 | 参数 | 返回 | 说明 |
|------|------|------|------|------|
| GET | `/student/{id}` | 路径参数 id | StudentDTO | 查询单个 |
| GET | `/student` | minAge, maxAge | List<StudentDTO> | 条件查询 |
| POST | `/student` | JSON body | Long (新ID) | 新增 |
| PUT | `/student/{id}` | 路径参数 + name, email | StudentDTO | 更新 |
| DELETE | `/student/{id}` | 路径参数 id | void | 删除 |

## 课程总结

恭喜你完成了 Spring Boot 学生管理系统的全部课程！

### 核心知识点回顾

```
Spring Boot 基础      → @SpringBootApplication, 自动配置
JPA 实体映射          → @Entity, @Table, @Column, @Id
Repository 数据访问   → JpaRepository, findBy, @Query
DTO 与 Converter      → Entity ↔ DTO 转换
Service 业务层        → 接口实现分离, @Transactional
Controller 控制器    → @RestController, RESTful API
枚举与统一响应        → enum, Response<T>
```

### 后续学习建议

1. **Spring Security** - 添加用户认证和权限控制
2. **Validation** - 添加参数校验（@Valid, @NotNull）
3. **全局异常处理** - @ControllerAdvice 统一异常处理
4. **Spring Cloud** - 微服务架构
5. **单元测试** - JUnit + Mockito
