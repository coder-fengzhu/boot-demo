# Spring Boot 学生管理系统 - 课程概览

本教程通过一个完整的学生管理系统项目，带你从零理解 Spring Boot 开发。

## 学习路径

| 课时 | 内容 | 核心知识点 |
|:---:|------|----------|
| 01 | Spring Boot 入门 | 项目搭建、启动类、自动配置 |
| 02 | 实体类与 JPA 映射 | @Entity、@Column、@Id |
| 03 | Repository 数据访问层 | JpaRepository、派生查询 |
| 04 | DTO 与 Converter | 数据传输对象、类型转换 |
| 05 | Service 业务逻辑层 | 依赖注入、事务管理 |
| 06 | Controller 控制器 | RESTful API、@RestController |
| 07 | 枚举与响应封装 | enum、泛型响应类 |
| 08 | 项目整合 | 完整调用流程梳理 |

## 项目效果

完成后你将掌握一套标准的学生 RESTful API：

```
GET    /student/{id}      # 查询单个学生
GET    /student?minAge=&maxAge=  # 条件查询
POST   /student           # 新增学生
PUT    /student/{id}      # 更新学生
DELETE /student/{id}      # 删除学生
```

## 准备环境

- JDK 17+
- MySQL 5.7+
- IDE (IntelliJ IDEA / VS Code)
